# 절감 잠재량 환산 — 수식·가정·출처

작성일: 2026-07-08 | 재현 스크립트: `scripts/06_savings_estimate.py` (모든 이벤트 통계는 데이터에서 직접 산출)

## 0. 환산 원칙

탐지된 비효율 구간의 총 전력량(약 32,000 kWh/5개월)을 절감액으로 환산하지 **않는다**. 해당 전력의 대부분은 정상 생산에 사용된 것이며, 절감 가능한 것은 품질 저하로 인한 **초과 손실분과 회피 가능한 비용**뿐이다. 이에 따라 ① 실측 기준, ② 예방 시나리오, ③ 파라미터식의 3단으로 구분해 제시한다.

## 1. 적용 계수 및 출처

| 계수 | 값 | 출처 |
|---|---|---|
| 산업용 전기요금 | 179 원/kWh | 한전 산업용 평균 판매단가 (2025 상반기), 2026년 1분기 동결 |
| 전력 배출계수 | 0.4173 kgCO₂/kWh | 환경부·온실가스종합정보센터, 2023년 국가 전력배출계수 (최신 공표) |
| 역률 요금 | 지상역률 90% 미달 1%p당 기본요금 0.2% 가산 | 한전 기본공급약관 제43조 |
| 전압 유지 범위 | 220V ± 13V (207~233V) | 전기사업법 시행규칙 제18조 |

## 3. 수식과 결과

### ① 실측 이벤트 기준 — 저역률 이벤트의 초과 선로 손실

동일 유효전력 전달 시 선로 손실은 전류 제곱에 비례하고, 전류는 역률에 반비례하므로:

```
초과손실(W) = α × P̄/3 × [ (PF정상/PF이벤트)² − 1 ]
절감량(kWh) = 초과손실 × 이벤트 총 시간
```

| 설비 | 이벤트 | 역률 (정상→이벤트 중앙값) | 초과손실 | 연간 환산 |
|---|---|---|---|---|
| 예비건조기 | 26,244건 (36.5h) | 88.1% → 63.1% | 28.5 W | 2.5 kWh |
| 6호기 | 25,670건 (35.7h) | 88.1% → 73.1% | 13.6 W | 1.2 kWh |
| **합계** | | | | **3.7 kWh/년 = 664원 = 1.5 kgCO₂** |

**해석**: 이벤트가 전체 시간의 약 1%(5초 단발)에 불과해 현재 직접 손실은 미미하다. 이 정직한 수치가 곧 "본 모델의 가치는 사후 절감이 아니라 조기 경보·예방에 있다"는 논거가 된다.

### ② 열화 상시화 방지 시나리오 — 조기 정비의 예방 가치 (연간)

만성적 이벤트는 콘덴서 뱅크·권선 열화의 전조이며, 방치 시 저역률이 상시화된다고 가정하면:

- (a) 초과 선로손실 상시화: 42.1W × 8,760h = **369 kWh/년 = 66,068원 = 154 kgCO₂**
- (b) 역률 페널티: 현재 청구 기준 역률(P·Q 산출)은 98.1%로 페널티가 없으나, 열화로 63%(이벤트 중앙값)까지 하락 시 기본요금의 5.4% 가산 = **291,600원/년**
- (a)+(b) = **연간 약 36만 원 + 154 kgCO₂의 발생을 조기 정비로 차단**

### ③ 전압강하(3호기) — 다운타임 회피 가치 (파라미터식)

관측: 5개월간 13,049건, 법정 유지범위(207V) 미달 포함, 최저 190.1V.

```
회피 가치(원/년) = n(방지된 정지 횟수/년) × h(평균 정지 시간) × L(시간당 생산손실, 원)
```

L은 공정·업종별 편차가 커서 임의 숫자를 넣지 않고 파라미터로 제시한다. 일반적으로 제조업 정전·순간전압강하 1건의 손실은 수백만~수억 원 규모로 보고되므로, ①·②를 압도하는 항목은 ③이라는 서술이 방어 가능하다.

## 4. 한계

- ①의 손실률 α, ②의 열화 시나리오, 계약전력·기본요금은 모두 가정이며 실제 공장 계약 조건으로 재산정 필요
- 본 데이터의 P·Q는 이벤트 중에도 정상값을 유지하므로(시뮬레이션 특성), ①은 물리 모델 기반 추정임
- 상별 역률 컬럼과 P·Q 기반 산출 역률(98.1%)이 불일치하는 것은 데이터 생성 특성으로, 요금 계산에는 P·Q 기반을 사용
